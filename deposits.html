Pledge

Engine requires sufficient balance to pay rewards for every stake, hence for each stake some amount would be reserved as pledge otherwise it wonâ€™t stake. For Lock it is full reward, for Stake it is unit reward. Note that if Stake reward exceeds pledge Engine would give maximum available currency but not more than reward.
Balance is a current treasury balance, the difference between Pledge and Balance is a volume of current Stakes.

Royalties

We impose our royalties on first launch and reward pool deposits.
  • 3 SOL on first launch
  • 10.0% for SOL
  • 10.0% for Stable Coins
  • 15.0% for Tokens
