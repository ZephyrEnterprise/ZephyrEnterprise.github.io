Getting Started

Tool required

  • Git is required to clone the repository, any version is suitable. Download from: https://git-scm.com/book/en/v2/Getting-Started-Installing-Git or via Brew on Mac Os or Linux
  • Node is required to run java script code, use any stable version from v.17. Download from https://nodejs.org/en/download/ or via Brew on Mac Os or Linux

    • Clone and Install

    Engine code would be run on your local machine, clone repository and install node_modules to launch. Run in your working folder:
    git clone https://github.com/ZephyrEnterprise/stake_engine
    Move to dashboard:
    cd stake_engine/engine_ui
    Install required modules with npm packet manager:
    npm install --force
    wait till process ends.

    Run

  • Before first launch put hash list into hashes.json and put it into ./stake_engine/engine_ui folder
    • To fund and sign transactions you need one of these virtual wallets installed as your browser extention
  • Phantom: https://phantom.app/
  • Slope: https://slope.finance/
  • Solfare: https://solflare.com/
    • Start dashboard:
    npm run serve
    Open http://localhsot:8080/ in your browser if not redirected automatically
