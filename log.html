Log

Click side bar to open the Log. All actions are presneted there including transaction signatures and errors. Note that Log refreshes each time you start Virtual Machine.

Data

All current files are stored in ./engine_ui/data folder. There you can find logs folder that stores separated logs with java script errors. When flushing files are cached into ./stake_engine/engine_ui/.cache folder by date and session.
Never give access to ./data and .cache folders to third party
